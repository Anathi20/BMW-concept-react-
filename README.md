# BMW Concept React Experience

A cinematic BMW landing experience built with React + Vite. It showcases immersive hero/video sections, a dealer locator, financing sandbox, and intelligence highlights inspired by BMW’s digital language.

## Live Preview
- https://bmw-concept-react.vercel.app/

## Tech Stack
- React 18 + Vite
- TailwindCSS & custom glassmorphism styling
- React Icons, GSAP-style scroll reveals (CSS-only)
- Vercel deployment

## Getting Started
```bash
npm install
npm run dev
npm run build
npm run preview
```

## Project Highlights
- Hero section with delayed glass overlay + CTA pair
- Cinematic video carousel with stat ticker
- Cars section including filters, dealer locator, and finance calculator
- News/Intelligence spotlight with animated tabs
- Responsive layout tuned for edge-to-edge presentation

## Deployment
This repo auto-deploys to Vercel (`npm run build`, output `dist`). Pushes to `main` trigger new previews and production releases.
