# BMW Concept React Experience

A polished BMW-inspired digital experience built with React + Vite, crafted to present a premium, portfolio-ready showcase of modern frontend design and interaction. The application combines immersive hero and video storytelling, interactive model exploration, a dealer locator, a finance sandbox, and intelligence-focused highlights, all tied together with smooth navigation, refined motion, and a responsive, production-ready layout.

## Live Preview
- https://bmw-concept-react.vercel.app/

## Tech Stack
- React 18 + Vite
- TailwindCSS & custom glassmorphism styling
- React Icons, GSAP-style scroll reveals (CSS-only)
- Vercel deployment

## Getting Started
```bash
npm install
npm run dev
npm run build
npm run preview
```

## Project Highlights
- Hero section with delayed glass overlay + CTA pair
- Cinematic video carousel with stat ticker
- Cars section including filters, dealer locator, and finance calculator
- News/Intelligence spotlight with animated tabs
- Responsive layout tuned for edge-to-edge presentation

## Deployment
This repo auto-deploys to Vercel (`npm run build`, output `dist`). Pushes to `main` trigger new previews and production releases.
